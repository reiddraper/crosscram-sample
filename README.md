# crosscram-sample

## Usage

```shell
git clone git@github.com:reiddraper/crosscram-sample.git
cd crosscram-sample

# play 100 rounds
lein run 100
# Winner Stats: {:bot-a 10, :bot-b 90}

# now edit bot_a.clj and/or bot_b.clj
# to have new bots play against each other
